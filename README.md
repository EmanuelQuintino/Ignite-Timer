# Ignite-Timer

![Ignite-Timer](./public/home-ignite-timer.png)

## Technologies

- `React.js`
- `TypeScript`
- `Styled-Components`
- `React-Router-Dom`

## Installation

```bash
# Clone repository
$ git clone https://github.com/EmanuelQuintino/Ignite-Timer.git

# Install dependencies
$ npm install

# Run project
$ npm run dev
```
